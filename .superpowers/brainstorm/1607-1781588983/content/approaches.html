How should Claude fit into the app?

Three approaches — pick the one that matches how you want to train

A

Export Bridge — Free

App handles all logging + visualizations locally. When you want coaching, tap "Copy to Claude" — the app bundles your health context + today's session + your note into a clean prompt. You paste it into your personal Claude (which already knows your medical history). Claude responds there.


Pros

  • Zero API cost, always
  • Your personal Claude keeps full medical history
  • No new context to manage

Cons

  • Still switching apps mid-workout
  • No "live" coaching flow in-app
B

In-App Claude Coach — ~$1–3/year ⭐ Recommended

App has a built-in Coach tab powered by Claude Haiku (the cheapest model). You store a health context file in the app — injury history, ortho notes, things to monitor — that gets sent with every call (cached, so you pay almost nothing for it). Pre-workout check-in, post-workout debrief, all in-app. Previous weights pulled automatically from your log.


Pros

  • Seamless — logging + coaching in one place
  • Coach sees your actual logged data automatically
  • Pre/post check-in flow built into the session
  • Cost: < $0.002 per workout session

Cons

  • Health context is a file you write/maintain
  • Separate from your personal Claude's memory
  • Requires API key (one-time setup)
C

Hybrid — In-App + Personal Claude Hand-off

App has an in-app Haiku coach and a "Send to my Claude" export. During workouts, use the in-app coach for quick check-ins (cheap, fast). After sessions, optionally export a full-context package to your personal Claude for deeper analysis — which knows your medical history.


Pros

  • Best of both — lightweight in-app + full depth when needed
  • Personal Claude stays authoritative on medical history
  • Flexibility — use whichever path fits the moment

Cons

  • Most complex to build
  • Two places Claude can respond — could feel fragmented
  • Still need API key + health context file for in-app piece

Cost reality check

Claude Haiku: $0.80/1M input tokens, $4/1M output. A typical workout check-in (health context + session data + your note + Claude's response) costs about $0.001–0.002. At 3 workouts/week, that's roughly $0.30–1.00/year. The health context is cached — you pay almost nothing for it after the first call.

Note: Regardless of which you pick, the app handles arms / legs / core as separate visualization categories, and the logging + history screens are identical.
